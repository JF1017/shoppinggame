# Grocery Math Cart

An interactive HTML and p5.js learning game that helps students practice addition with multiple addends through daily grocery shopping.

## Run Locally

Start a simple local web server from this folder:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://127.0.0.1:8000/
```

## Project Files

- `index.html` contains the page structure, styles, and controls.
- `sketch.js` contains the p5.js game logic and cart drawing.
- `images/` contains the item art used in the shopping game.
